# CONTEXTO DEL PROYECTO: R3 CONSULTORES - WEB Y APP

## 1. Objetivo Principal del Proyecto
Crear una plataforma digital integrada para "R3 Consultores". El sistema se divide en dos partes:
1. **Página Web (Landing/Informativa):** Un sitio moderno, intuitivo y funcional orientado a la captación de clientes. Mostrará los servicios, permitirá agendar citas y facilitará el contacto directo.
2. **Aplicación Web (Portal de Clientes):** Embebida o accesible desde la página web, donde los clientes podrán ingresar sus datos y gestionar sus obligaciones tributarias y contables.

## 3. Portafolio de Servicios (R3 Consultores)
La web debe destacar claramente los siguientes servicios:
* Asesoría Contable, Financiera y Tributaria.
* Declaraciones de impuestos: IVA, Renta, Retenciones en la fuente.
* Elaboración de información Exógena.
* Consultoría especializada.
* Llevanza de contabilidades (Outsourcing contable).
* Trámites legales y comerciales: Registro de RUT, trámites ante Cámara de Comercio.
* Organización y estructuración de empresas.
* Implementación de software contable (Especialistas en World Office).

## 4. Funcionalidades Clave Requeridas (Fase Web)
* **Navegación Intuitiva:** Menú superior claro (Inicio, Servicios, Nosotros, Contacto, Portal Clientes).
* **Agendamiento de Citas:** Un apartado o integración para agendar reuniones con un experto contador.
* **Botón de WhatsApp:** Flotante y visible en todo momento para atención rápida.
* **Ingreso a la App:** Un botón claro (ej. "Portal Clientes" o "Ingresar") que lleve a la aplicación web.

1. **Cero Suposiciones:** Si falta un dato técnico, estructura de base de datos o preferencia, **PREGUNTA** antes de escribir código.

3. **Desarrollo Modular:** Entrega soluciones paso a paso. No intentes programar toda la aplicación en una sola respuesta.
4. **Enfoque en el Negocio:** Mantén siempre el contexto de que es una firma de contadores públicos en Colombia; el lenguaje debe ser profesional, claro y orientado a la confianza financiera.
